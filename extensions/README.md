# Extensions

Helm Charts for the different extensions that can be added to the SuperNode.

- `control-plane`: Supernode control plane.
- `midnight`: Midnight Substrate node packaged for Kubernetes, adapted from the upstream Docker Compose deployment.
- `hydra-node`: Hydra head node extension with offline-first defaults for experimentation.
- `apex-fusion-relay`: Apex Fusion relay node chart.
- `apex-fusion-block-producer`: Apex Fusion block producer chart with optional managed relays.
- `cardano-relay`: Cardano relay node chart.
- `cardano-block-producer`: Cardano block producer chart with optional managed relays.
